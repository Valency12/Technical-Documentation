Python

Introduction

Python es un lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning (ML). Los desarrolladores utilizan Python porque es eficiente y fácil de aprender, además de que se puede ejecutar en muchas plataformas diferentes. El software Python se puede descargar gratis, se integra bien a todos los tipos de sistemas y aumenta la velocidad del desarrollo.

Variables

Una variable es un nombre que se asocia con un valor que se almacena en la memoria. Las variables actúan como etiquetas para referirse a los datos que queremos manipular o utilizar en un programa. Su sintaxis es variable_name = value. Ejem:

x = 10
y = "Hello, World!
Data Types

Los tipos de datos son las categorías en las que se agrupan los valores que las variables pueden almacenar . Cada tipo de dato define el tipo de operaciones que se pueden realizar sobre los valores y cómo se almacenan en la memoria. Los tipos son Integer, Float, String, Boolean, List, Tuple, Set, Dictionary. Ejem:

#Integer x = 12
#Float y = 12.89
#String saludo = "Hello Everyone!"
#Boolean state = true or false
#List arr = [1, 2, 3]
#Tuple arr = ('A', 'B', 'C')
#Set numbers = {2, 4, 6, 8}
#Dictionary dic = {nombre: "Juan", edad: "12"}
Expressions

Las expresiones son combinaciones de valores, variables, operadores y llamadas a funciones que se evalúan y devuelven un valor. Una expresión puede ser tan simple como un número o una variable, o tan compleja como una combinación de varias operaciones y funciones. Siguiendo el orden de las operaciones (PEMDAS) parentesis( ), exponentes, multiplicacion *, division /, suma + y resta - . Ejem:

result = 25 * 5 + 100
Operators

Los operadores son símbolos que realizan operaciones sobre valores u operandos. Dependiendo del tipo de operación que realicen, Se pueden clasificar en varias categorías:
Aritmeticos +, -, *, /, %, **, //
Asignacion =, +=, -=, *=, /=, %=, //=, **=
Comparacion ==, !=, >, <, >=, <=
Logicos and, or, not

#Aritmeticos
a = 10
b = 3
print(a + b) # 13
print(a - b) # 7
print(a * b) # 30
print(a / b) # 3.333
print(a % b) # 1
print(a ** b) # 1000
print(a // b) # 3

#Asignacion
x = 5
x += 3 # x = x + 3 → x es ahora 8

#Comparacion
a = 5
b = 3
print(a == b) # False
print(a != b) # True

#Logicos
a = 5
print(a > 3 and a < 10) # True
print(a > 10 or a == 5) # True
print(not(a == 5)) # False
Statements

Las sentencias condicionales se utilizan para tomar decisiones en el código. Nos permiten controlar el flujo de nuestro programa. En Python, usamos principalmente las sentencias if para ejecutar código cuando una condición es verdadera, elif para múltiples condiciones y if-else se utiliza cuando quieres ejecutar código diferente para condiciones diferentes.

#if
num = 5
if num > 0:
print("El número es positivo.")

#if-elif-else
temperature = 25
if temperature < 0:
print("¡Hace frío!")
elif temperature >= 0 and temperature < 15:
print("Hace frío.")
elif temperature >= 15 and temperature < 25:
print("Es templado.")
else:
print("Hace calor.")

#if-else
num = -3
if num > 0:
print("El número es positivo.")
else:
print("El número no es positivo.")
Loops

Los loops en Python, o bucles, son estructuras que permiten ejecutar un bloque de código repetidamente hasta que una condición específica se cumpla. Existen dos tipos principales de loops en Python: for y while.Ejem:

#for
Recorrer una lista de frutas.
fruits = ["manzana", "banana", "cereza"]
for fruit in fruits:
print(fruit)

#while
count = 0
while count < 5:
print("Cuenta:", count)
count += 1
Function

Una función es un bloque de código reutilizable con nombre que realiza una tarea específica. Las funciones se definen y se invocan utilizando su nombre, y pueden aceptar parámetros. Sintaxis: def nombre_funcion(parámetros): Ejem:

def greet(nombre):
print("Hola, " + nombre)
Llamada a la función: greet("Alice")
Salida: Hola Alice
Libraries

Las librerías en Python son colecciones de módulos que contienen funciones y herramientas reutilizables. Una librería te permite evitar escribir código desde cero para tareas comunes, ya que encapsulan funcionalidades listas para ser usadas. Entre algunas de sus librerias se encuentras: math, NumPy, Pandas, Seaborn, TensorFlow, Keras y muchisimas mas.
